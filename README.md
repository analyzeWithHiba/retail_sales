Retail Sales Data Analysis

Description:
This project analyzes retail sales data to uncover insights about customer behavior, product performance, and sales trends. Using Python (Pandas, Seaborn, Matplotlib), the analysis includes data cleaning, aggregation, visualizations, correlation analysis, and identification of top-performing categories and customers. Key insights include revenue patterns, high-value customers, discount usage, and premium product categories, supporting data-driven business decisions.

Technologies Used: Python, Pandas, Seaborn, Matplotlib, Jupyter Notebook

Key Features:

Total and quantity sales analysis by category

Top 10 items and customers identification

Monthly sales trend visualization

Correlation heatmap of numeric variables

Discount usage analysis

Outlier detection and price vs spending insights
